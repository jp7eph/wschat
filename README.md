# wschat
WebSocket Chat
送信した文字列の末尾に [HOGE] を付けてサーバからレスポンスされます．

## How to Use

1. [CONNECT] を押す
1. テキストボックスに好きな文字列を書く
1. [SEND] を押す
1. レスポンスが帰ってくる（青文字）
1. [DISCONNECT] を押す
